POIJU.IO

Welcome to POIJU.IO developer documentation. POIJU.IO is a free open source service for nautical charts for Finland sea and lake areas.

Want to use POIJU.IO?

Go to www.poiju.io using a browser of your choice on laptop, desktop, mobile device or tablet. Remember to allow location services on your device. POIJU.IO is also available on iOS AppStore.

For developers:

POIJU.IO is open source project and available on GitHub. Want to add new features? Do you have an awesome idea similar to POIJU.IO - for example an app for outdoor people? Now you can do it!

Setting up local environment:

To get started clone POIJU.IO repository:

git clone https://github.com/iaue/poiju.io.git

For running POIJU.IO locally NODEJS, NPM and BOWER are required. More information: https://docs.npmjs.com/getting-started/installing-node

Go to POIJU.IO directory and run following commands:

npm install

bower install

These commands will install all the POIJU.IO server and front end dependencies.

Running POIJU.IO locally:

To start POIJU.IO server locally type in following command:

nodejs start

This will start POIJU.IO server on your computer. Open browser and go to https://localhost:5555. For http version use http://localhost:5000
